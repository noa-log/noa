# 包装错误
Noa 提供`errors`包来对错误进行包装，以便提供更好的错误上下文信息。包装后的错误可以在打印时自动附加堆栈信息。

## 手动包装错误
```go
package main

import (
    baseErrors "errors"
    "github.com/noa-log/noa"
    "github.com/noa-log/noa/errors"
)

func main() {
    // 创建一个新的日志实例
    logger := noa.NewLog()

    // 手动包装错误
    err := errors.Wrap(baseErrors.New("一个错误示例"), 2)
    
    // 打印包装后的错误
    logger.Println(noa.ERROR, "Test", err)
}
```
